#### Suivez https://viability-theory.org/node/22/ pour cloner le dossier VIABLAB  ####

#### Installer cmake-3.13.5 ####
	#### Telecharger cmake-3.13.5-Darwin-x86_64.tar.gz sur https://github.com/Kitware/CMake/releases/tag/v3.13.5 ####

	#### Verifier le nom de cmake : ####
	sunxt@SunXTdeMacBook-Air % cd ~/Downloads
	sunxt@SunXTdeMacBook-Air Downloads % ls ~/Downloads | grep cmake
		## répond ##
		cmake-3.13.5-Darwin-x86_64.tar

	#### Accédez au répertoire de téléchargement et décompressez le fichier : ####
	sunxt@SunXTdeMacBook-Air Downloads % tar -xvf cmake-3.13.5-Darwin-x86_64.tar

	#### Déplacez-le dans le dossier viablab : ####
	sunxt@SunXTdeMacBook-Air Downloads % mv cmake-3.13.5-Darwin-x86_64 ~/Desktop/viablab/cmake

	#### Verifier la version: ####
	sunxt@SunXTdeMacBook-Air Downloads % ~/Desktop/viablab/cmake/CMake.app/Contents/bin/cmake --version
		## répond ## 
		cmake version 3.13.5

	#### Ajouter la commande cmake au PATH global pour l'utiliser plus facilement : ####
	sunxt@SunXTdeMacBook-Air Downloads % nano ~/.zshrc
		#### Ajoutez la ligne suivante à la fin du fichier : ####
		export PATH="$HOME/Desktop/viablab/cmake/cmake-3.13.5-Darwin-x86_64/CMake.app/Contents/bin:$PATH"
		#### Sauvegardez et quittez (appuyez sur Ctrl + X, puis Y, et enfin Entrée). ####
		#### Appliquez les modifications : ####
		sunxt@SunXTdeMacBook-Air Downloads % source ~/.zshrc

#### Vérifiez que le dossier VIABLAB/build est vide ####
	sunxt@SunXTdeMacBook-Air Downloads % cd ~/VIABLAB 
 	sunxt@SunXTdeMacBook-Air VIABLAB % rm -rf build
	sunxt@SunXTdeMacBook-Air VIABLAB % mkdir build
